# Tic-Tac-Toe (React + TypeScript + Vite)

A modern, responsive, and clean Tic-Tac-Toe game built with React, TypeScript, and Vite.

## Features
- **Local Two-Player Gameplay**: Play pass-and-play with custom player names.
- **Turn Indicator**: Clear active highlights showing whose turn it is.
- **Interactive Scoreboard**: Tracks player wins and ties across multiple rounds.
- **Sleek Layout**: A clean Slate Gray dark mode UI.
- **Winner Highlights**: Pulsing victory animations styled to match the winning token's color (Green for X, Red for O).
- **Game Controls**: 
  - **Next Round**: Start a new round while retaining scores.
  - **Reset Match**: Reset the game board and wipe scores back to zero.

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation
1. Clone the repository
2. Navigate into the directory and install dependencies:
   ```bash
   npm install
   ```

### Development Server
Run the local dev server:
```bash
npm run dev
```

### Production Build
Compile and bundle the project for production:
```bash
npm run build
```
